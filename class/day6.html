Control flow: Loops

Looping is the act of repeating a block of code based on a given condition. When the conditions ceases to be truthy, the loop stops.

The four loops in Javascript are:

  • While loops
  • Do / while loops
  • for loops
  • for / in loops

While loops work by evaluating a condition, if the condition is true, the loop "fires", executing all code it contains. The function will keep looping back to the top and executing until the condition becomes false, then loop stops.
One important concept with while loops is that you must write the code that will make the condition false, otherwise you get an infinite loop. (code has no break point and continues to evaluate)

var a = 0; // assign a variable named a with an initial value of 0
while (a < 10) { // designation of "while loop" with condition to be evaluated for truthy
console.log(a); // during each iteration through the loop that returns truthy console.log (a)
a++; // short hand to increment by 1 each time the loop has iterated through
}

Do / While loop is less commonly used, however it's a useful tool to have once you understand how it differs from the while loop.
The do / while loop will always run at least once, then it evaluates its condition. If the condition is still true, it will continue the loop.

var i = 1; // assign a variable named i with an initial value of 1
do { // designation of "do " statement
console.log(i); // First console.log (i) with out checking for truthy. then continue to console.log (i) on each iteration that returns truthy.
i++; // short hand to increment by 1 each time the loop has iterated through
} while (i < 10); // Code has already ran once before the while statement checks condition for truthy.

for loop, the most common loop we'll use in class and that you will encounter as a programmer.
A for loop looks like this:

for (statement1; statement2; statement3) {
// code block to be executed
}

The 3 statements used in the loop define the conditions under which the loop runs.

  • Statement 1 is executed before the loop (the code block) begins. This allows us to set up or initial state.

  • Statement 2 defines the condition for running the loop

  • Statement 3 is executed each time after the loop has executed. This is where we update our conditional variable. For example, increment or decrement a variable.
for (var i = 0; i < 10; i++) {
// Statement 1 var i is set to initial state of 0
// Statement 2 i < 10 defines the condition for running the loop.
// Statement 3 i++ executes each time after the loop has executed. Incrementing i by 1
console.log(i);
}

JavaScript provides two ways of controlling the flow of a loop, the continue and break keywords.
Continue will immediately reset the loop to the next iteration, and continue on with the loop. It's basically just saying, disregard the rest of this loop and go to the next loop.

var i = 0;
while (i < 10) {
if (i === 5 || i === 7) {
i++;
continue; // The above while loop condition excludes i when value is set to both 5 and 7. continue allows the code to skip these values and loop through the rest of the code.
}
console.log(i);
i++;
}

The break keyword will, when encountered, immediately stops the loop AND all subsequent loops.

for (var i = 0; i < 10; i++) {
console.log(i);
if (i === 6) {
break;} // immediately stops loop when i equals 6
}

for / in statement loops through the properties of an object:
A for / in loop looks like this:

var person = {fname:"John", lname:"Doe", age:25};
// Object person with named vale pairs or properties fname: "john", name:"Doe", age:25
var text = "";
// variable named text assigned empty quotes
var x;
// variable declared x with undfined value
for (x in person) {
// for loop assigning value to variable x
text += person[x];
// variable text iterates through object person assigning values to variable x
}

Looping & Arrays

So all our work with arrays up to this point has been retrieving array indexes one at a time using a numbered bracket notation, IE fruits[3]. This is useful, but what if your application requirements were such that you needed to do something to every element in the array?
This where were the power of loops really becomes apparent. A for loop is a perfect mechanism for iterating over each element in an array.

var fruits = ["cherry", "apple", "peach", "plum"];
for (var i = 0; i < fruits.length; i++) {
// for loop statement 1 sets the initial value of i
// for loop statement 2 defines the condition for running the loop.
// for loop statement 3 executes each time after the loop has executed.Incrementing i by 1
console.log(fruits[i]);
}
