Variables

Variables are containers for storing data values.
Format to "Declare" a variable:

var name = 'Jason';

Some notes for naming variables.


  • Javascript names (identifiers) must be unique
  • Names can contain letters, digits, underscores, and dollar signs.
  • Names must begin with a letter
  • Names can also begin with $ and _
  • Names are case sensitive (y and Y are differnt variables)
  • Reserved words such as JavaScript keywords cannot be used.

Comments

Comments don't do anything in the code.
They are specifically for documentation of code to help explain its functionality.
Comments are written in two ways.
Firstly a single line of comments must be started with two forward slashes //
Lastly a multi line comment starts with a forward slash followed immediately by an asterisks /*
(Multiple lines of code )
Followed by an asterisks and another forward slash */

// initialize my variables
var name = 'Jason';
var a = 1;
var b = '2';
/* initialize my variables
"name"
"a" and "b" */
var name = 'Jason';
var a = 1;
var b = '2';

Data Types

String
Strings are used to store and manipulate text.
Strings can be declared inside both single ' ' and double " " quotes.

var text = 'This is a string in JavaScript';
var numberString = 'This is a 2nd string';

Number
JavaScript has one kind of number type which can be written with or without decimals.

var n1 = 123;
var n2 = 123.4;

Large or small numbers can be written with scientific (exponent) notation:

var largeNumber = 4e5; // 40000
var smallNumber = 12e5; // .00012

1 + 2 = 3. What about "1" + "2"? (12)
Boolean
Booleans can represent one of two values: true or false;

var a = true;
var b = false;

Objects
Objects are a complex variable type that can contain many variables and functions in one place. These variables and functions are usually refered to as properties and methods.
Javascript objects often are used to represent real life objects.
Properties usually represent information and values in the object.
Methods usually represent actions the object can take.

var a = {};
var color = ["red", "green", "blue"];

Null
A null is "nothing"
It is used to represent something that doesn't exist.

var example = null;

Undefined
A variable without a value is undefined.
A variable can be emptied by setting it's value to undefined.

var name; // value is undefined.
var a = 4;
a = undefined; // Value is undefined, a is now empty

Significand, base, exponent
Computers usually use IEEE 754 standard for Floating Point Arithmetic

Many programming languages have many number types, such as integer, float, decimal, long...
JavaScript has only one number type: double precision floating point
This format stores numbers in 64 bits, where the number (fraction) is stored in bits 0 to 51, the exponent in bits 52 to 62, and the sign in bit 63:

Value (aka Fraction/Mantissa) Exponet Sign
52 bits (0 - 51) 11 bits (52 - 62) 1 bit (63)
