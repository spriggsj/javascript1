Inline JavaScript

Example: In a HTML document the onclick attribute can be used to add some functionality to a DOM node.(This is not the recommended way of attaching functionality to your DOM elements.) This is considered inline since the code that JavaScript will execute is inside the node itself. It lives in the page, thus it is considered inline.

Inline Script Tag

Example 2: The next way is another type of inline JavaScript, in that it's JavaScript that is executed in the context of the html page. IE the code itself still lives inside the html page, but in its own script tag.
<script>Code here</script>

Script tag with Src

Example 3: The prefered way of including scripts in your page is similar to the previous way, except instead of putting your JavaScript inline you specify an external JavaScript file that the browser will include in your page and execute.
<script type="text/javascript" src="app.js"></script>

Execution of a JS file

JavaScript always executes from top to bottom. So if you include two JavaScript files the browser will first download the top file execute it line by line, then download the next file down and execute it line by line. In the case where you want to utilize something defined in one JavaScript file by another JavaScript file make sure the file that initializes is positioned above in the order of operation.

Truthy / Falsy

Certain values in JavaScript, when evaluated, are considered "falsy" or "truthy"
The following values are always falsy:
  • false
  • 0 (literal number zero)
  • "" empty string
  • null
  • undefined
  • NaN (Not a number)
All other values are truthy, including "0" (zero in quotes), "false" (false in quotes), empty functions, empty arrays, and empty objects.

Intrinsic Objects

String Object
Strings can be treated as objects in JavaScript. This gives us access to special built-in functions to use on strings.
length(), substring(x,y), toUpperCase(), …
var strA = "0123456789";
// this code sets strB to "456"
var strB = strA.substring(4,7);
var str1 = "airplane";
// set str2 to "AIRPLANE"
var str2 = str1.toUpperCase();

Math Object


Contains functions to perform mathematical operations and conversions
Math.PI, Math.round(), Math.sqrt(), …
Date Object
The Date object can be used to represent and compare dates, as well as get the current system time.
var today = new Date();
var thisYear = today.getFullYear();

Number Object


The number object contains special numeric constants that are important to JavaScript.
Number.MAX_VALUE, Number.MIN_VALUE, Number.NaN,
Number.POSITIVE_INFINITY, Number.NEGATIVE_INFINITY
